# Фоновая музыка

Положите **один** из файлов в эту папку (сайт попробует все варианты):

- `background.mp3` (рекомендуется)
- `background.mpeg`
- `background.mp4`
- `background.wav`

Путь: `public/music/background.mp3`

Музыка включается при загрузке; если браузер блокирует — при первом касании экрана или по кнопке ♪ внизу справа.
